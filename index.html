C++中const的各种用法
发表于2024-11-17|学习笔记
在 C++ 中，const 关键字用于声明常量，指示某个变量的值不能被改变。它是 C++ 中的重要语法工具，广泛应用于不同的编程场景中，能够提高代码的可维护性、可读性和安全性。以下是 const 的全部使用场景及详解： 1. 常量变量（Constant Variables）最常见的 const 用法是声明常量变量，它确保变量在初始化之后不能修改其值。 1const int MAX_SIZE = 100; 在此例中，MAX_SIZE 在程序中被视为常量，编译器会阻止你尝试修改它。 注意： 常量变量的初始化必须在声明时进行，并且之后不可修改。 2. 指针与常量指针const 可以与指针一起使用，分别定义常量指针、常量指向的对象，以及两者的组合。 2.1 常量指针（Pointer to const）1const int* ptr; 这里 ptr 是一个指向 const int 的指针，意味着通过 ptr 不能修改 *ptr 的值，但 ptr 本身是可以指向不同的地址的。 1234int a = 10, b = 20;const int* ptr = &a; //...
